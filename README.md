# File Order Application

A simple Java Swing application that organizes files in a selected directory by name or size.

## Features

- **Browse** - Select any folder on your system
- **Order Files** - Sort files by:
- **Alphabetic Order** - Sorts files A-Z by name
- **Numeric Order** - Sorts files by size (smallest to largest)
- **Clear** - Reset all fields and results
- Displays total file count with formatted file sizes (bytes/KB/MB)

## How to Use

1. Click **Browse** to select a folder
2. Choose sorting method: **Alphabetic Order** or **Numeric Order**
3. Click **Order** to sort and display files
4. Use **Clear** to start over

## Requirements

- Java 21 or higher
- Maven (for building)
